# praczero
Introduction to python 

Aim

To learn how to use the print() function in Python to display strings, numbers, lists, and variables.

Theory

The print() function in Python is used to show output on the screen. It can print different types of data such as text, numbers, and lists. Variables store values, which can be printed later. The sep="\n" option is used to print multiple values on separate lines.

Conclusion

This program helps us understand how the print() function works in Python and how it is used to display different kinds of data clearly.
